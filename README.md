# Multiplayer Tic-Tac-Toe

A production-ready, server-authoritative multiplayer Tic-Tac-Toe game using Nakama (TypeScript) and React/Vite (Vanilla CSS).

## Project Structure

- `frontend/`: React + Vite client.
- `backend/`: Nakama game server extensions written in TypeScript.
- `project-info/`: Contains instructions, assignment details, and phase documentation.

## Setup Instructions
Please refer to the detailed phase documentation located in `project-info/implementation/`.
